# 📝 Fullstack Todo App

A fullstack Todo application built with modern technologies.

---

## 🚀 Tech Stack

### 🔹 Frontend
- React
- Material UI (MUI)
- Axios

### 🔹 Backend
- Laravel
- REST API
- MySQL

---

## ✨ Features

- Add new todos
- Edit todos with smooth animation
- Delete todos with animation
- Clean modern UI
- Full CRUD operations
